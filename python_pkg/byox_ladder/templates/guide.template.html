The Build-Your-Own-X Guide Ladder

Every individual tutorial from the build-your-own-x collection — 0 guides across 30 categories — placed on a difficulty ladder. Search, filter by tier / language / category, and click any guide to open it.

How difficulty is assigned: each guide inherits its category's tier (a Regex-Engine guide is easier than an OS guide, whatever the language). Language only nudges the order within a tier — lower-level languages (C, Rust, Asm) sit a little higher than Python or JS. The Unsorted group is the repo's own uncategorized bucket; its guides span all levels, so they aren't ranked.
